Profile Picture from p1

When you (Pepe) open this page, it will load your profile picture from p1 using an img tag (Method 1) and attempt to send it to a webhook using fetch (Method 2). The login page for p1 will open automatically in 5 seconds to ensure a valid session.

Method 1: Using <img> tag

This uses a simple img tag to load the image. The browser automatically sends the session cookie if available.

Method 2: Using fetch

This uses JavaScript fetch to load the image as a Blob, display it, and send it to webhook.site. It includes credentials to send the session cookie.
